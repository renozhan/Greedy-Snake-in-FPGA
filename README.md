# 不會變長的貪吃蛇
1. 8x8 LED 矩陣用來顯示遊戲畫面。
2. 蛇的長度固定為2格，畫面上5秒會出現一個食物，新食物出現時舊食物消失。
3. 綠色食物加1分，藍色食物加2分，紅色食物扣2分。（玩家分數在2分以上時才會出現紅色食物）
4. 七段顯示器用來顯示剩餘秒數、LED陣列用來計分，2分鐘（120秒）內獲得16分即獲勝。
5. 成功時會發出兩聲短的嗶聲，失敗時發出一聲長嗶聲。
